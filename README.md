## Amazon Clone Website

This project is a front-end clone of the Amazon homepage built using **HTML and CSS**. It replicates the layout and design of Amazon's main page including the navigation bar, product sections, category cards, and footer.

## Features

* Amazon-style **navigation bar**
* **Hero section** with promotional banner
* **Product category cards**
* **Responsive layout**
* **Amazon-like footer with multiple sections**
* Clean and organized **HTML & CSS structure**

## Technologies Used

* HTML5
* CSS3
* Font Awesome (for icons)

## Purpose

This project was created to practice **front-end web development** and improve skills in **HTML layout, CSS styling, and responsive design**.

## Future Improvements

* Add **JavaScript functionality**
* Make the website **fully responsive**
* Implement **product pages**
* Add **shopping cart functionality**
